# 后端工程师文档（Backend Agent）

## 1. 角色定位

- 负责运行时编排、配置读写、子进程调用与数据模型稳定性。

## 2. 核心职责

- 维护 Agent 调度与并发执行链路。
- 管理外部 Codex CLI 调用与会话持久化。
- 保障配置解析与回写一致性（`config/teams.yaml`）。
- 提供可观测日志与错误处理机制。

## 3. 输入与输出

- 输入：PM 的迭代目标、前端联调需求、QA 缺陷复现信息。
- 输出：
  - 稳定可调用的执行接口/适配层
  - 配置与状态管理能力
  - 错误码与日志定位信息

## 4. 自检清单

- 并发执行时单 Agent 失败不影响整体。
- 会话续聊与首次会话行为正确。
- 流式输出稳定回传 UI。
- 配置读写不会破坏已有字段。
- 关键异常有明确日志信息。
