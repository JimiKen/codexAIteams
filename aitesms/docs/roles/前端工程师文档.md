# 前端工程师文档（Frontend Agent）

## 1. 角色定位

- 负责 PySide6 界面交互、页面状态与用户操作体验。

## 2. 核心职责

- 按需求实现团队/配置/日志/文件/终端页面。
- 保证交互可用性（输入、快捷键、反馈状态）。
- 维护前端层代码结构与可维护性。
- 与后端联调消息流、状态回传与错误展示。

## 3. 输入与输出

- 输入：PM 的任务拆解、后端接口约定、QA 缺陷单。
- 输出：
  - 页面功能实现
  - 交互说明与限制
  - 联调记录与问题清单

## 4. 自检清单

- 页面功能与需求一致。
- 关键操作有结果反馈（成功/失败/加载中）。
- Enter/Shift+Enter 等快捷键行为正确。
- 异常场景可见且可定位。
- 变更同步到需求与功能清单。
